Instructions setting values on Datapath
******************************
ExtOp: zero, sign
Memwr: 1 => write memory
ALUsrc: 0=> regB;
    1 => immed
MemtoReg: 0 => ALU; 1 => mem
ALUctr: "ADD", "SUB", "OR"
    log base 2 of number of supported operations
RegDst: 0 => "rt"; 1 => "rd"
RegWr: 1 => write register
