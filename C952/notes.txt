Write policy 
******************************
Cache hit:
    - write through
        need buffer
    - write back
        only tries to update cache 
        less traffic (since updates only on way out)
        bring in clean line into memory, then update piece then set to dirty

Cache miss:
    - no write allocate:
        only write to main memory
    - write allocate 
        bring in whole line
        fetch into cache
